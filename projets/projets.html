Pokedex App

Le Pokédex est un projet d'étude conçu pour explorer les appels API dans une application mobile native. Il utilise une API publique pour récupérer une liste de Pokémon, accompagnée de leurs informations détaillées telles que les types, les capacités et les statistiques. Ce projet permet de mettre en pratique la gestion des données dynamiques, en affichant des informations qui peuvent être mises à jour en temps réel. L'application présente un affichage structuré avec une navigation fluide et une interface utilisateur qui facilite l'accès aux données, même pour un grand nombre de Pokémon. À travers ce projet, j'ai approfondi l'intégration d'API externes, le traitement des données JSON et l'affichage optimisé dans une application mobile, tout en assurant une expérience utilisateur engageante et réactive.

100%
Retrofit API Json XML Picasso AndroidSDK

Instagram Remake App

La réplique de l'application Instagram est un projet d'étude conçu pour explorer les fonctionnalités avancées d'Android. Il permet de simuler une application de réseau social avec des éléments clés tels que les profils d'utilisateurs, les publications, et les interactions entre utilisateurs. Ce projet met l'accent sur la gestion des fragments pour une navigation fluide, l'utilisation des RecyclerViews pour afficher les données de manière dynamique et performante, ainsi que l'intégration des requêtes API via Retrofit pour récupérer les données en temps réel. Il explore ainsi la gestion de l'interface utilisateur complexe et la communication réseau dans une application mobile native.

Progrès du projet
100%
Kotlin XML Gradle Retrofit Picasso SharedPreferences RecyclerView Fragments

Athenas App

Athenas est une plateforme éducative que j'ai développée pour offrir aux élèves un espace où ils peuvent partager leurs travaux scolaires de manière anonyme avec leurs pairs. Les utilisateurs peuvent soumettre leurs projets, recevoir des retours constructifs de leurs camarades, et accumuler des récompenses en fonction de l'interaction et de la qualité de leur travail. Ce projet met en lumière la gestion des données utilisateur, l'implémentation des systèmes de récompense, ainsi que le développement d'une interface fluide et intuitive, tout en garantissant l'anonymat des utilisateurs pour encourager un partage sans jugement. Il explore également les aspects de sécurité et de gestion des utilisateurs dans une application éducative.

Progrès du projet
65%
Kotlin XML Gradle
